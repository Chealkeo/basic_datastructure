## 题目
请判断一个链表是否为回文链表。

示例 1:

输入: 1->2
输出: false
示例 2:

输入: 1->2->2->1
输出: true
## 思路
思路一：利用快慢指针走到中间结点slow处，
```python
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        if not head or not head.next: return True
        #划分为两截儿
        fst,slw = head,head
        while fst and fst.next:
            flag = slw
            slw = slw.next
            fst = fst.next.next
        fst = head
        flag.next = None
        #反转后半截儿，把后半截儿的最后一个结点即反转后的第一个结点赋值为cur，开始和fst指针指向的结点进行比较
        prev = None
        while slw:
            tmp = slw.next
            slw.next = prev 
            prev = slw
            slw = tmp
        cur = prev
        while fst:
            if fst.val==cur.val:
                fst=fst.next
                cur=cur.next
            else:
                return False
        return True
```
